## Чеклист

- [x] ICP: российские экспедиторы, 5–50 чел.
- [x] 3 тарифа: Start / Pro / Enterprise
- [x] MVP scope: лиды + заказы + контрагенты (Start); + документы + финансы (Pro)
- [x] Traklo: только Enterprise SKU
- [ ] One-pager для продаж (PDF/Notion)
