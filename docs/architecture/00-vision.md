# SaaS CRM для экспедиторов — видение

## Миссия

Дать любой экспедиторской компании (не только Автоальянс) CRM, заточенную под логистику: **лид → заказ → документы → финансы → диспозиция**.

## Проблема

- Generic CRM (amo, Битрикс) не понимают маршрут/груз/перевозчика
- 1C — тяжёлый, не для оперативной работы продавца
- Excel — нет единого контура, нет графика оплат

## Решение

Vertical SaaS на базе проверенной внутренней CRM:
- Воронка лидов с бизнес-процессом
- Мастер заказа (маршрут, груз, финансы, исполнители)
- График оплат с installments
- DOCX-печать с QR-верификацией
- RBAC по ролям и подразделениям

## Целевой клиент (ICP)

| Параметр | Значение |
| --- | --- |
| Отрасль | Грузоперевозки / экспедирование |
| Размер | 5–50 сотрудников |
| География | Россия (day-1) |
| Боль | Разрозненные Excel + почта + 1C |
| Готовность | Облако, subdomain, trial |

## Конкурентное отличие

| Мы | Они |
| --- | --- |
| Логистический wizard | Generic deal pipeline |
| График оплат per order | Счета без привязки к рейсу |
| DOCX + QR verify | PDF без верификации |
| KPI продавцов | Нет отраслевой аналитики |
| AI command bar (Pro) | Нет MCP/API для AI |

## Не делаем

- Generic CRM для всех отраслей
- White-label fork на клиента
- Traklo/mobile в base tier
- Feature parity с внутренней CRM в MVP
