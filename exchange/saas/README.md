# Exchange SaaS sync folder

This directory mirrors the requested Windows sync folder:

```text
C:\Sync\Yandex.Disk\Exchange\saas
```

Use it for files that must be shared between agents, local Windows work, and
the SaaS repository:

- handoff notes;
- indexes;
- discovery reports;
- architecture notes;
- migration notes;
- prompts for future agents.

When working on Windows, copy or synchronize this directory's contents to:

```text
C:\Sync\Yandex.Disk\Exchange\saas
```

The current cloud environment cannot write to that Windows path directly.
